# Bolnica Zdravlje+ 

> Projekat razvijen tokom 2025. godine kao univerzitetski projekat iz predmeta **Razvoj višeslojnih aplikacija (RVA)** na Fakultetu tehničkih nauka u Novom Sadu.

> Naknadno postavljen na GitHub radi prezentacije rada i korišćenih tehnologija.

Aplikacija služi za evidenciju bolnica, odeljenja, dijagnoza i pacijenata. Backend je napravljen u Spring Boot-u, a frontend u Angular-u sa Angular Material komponentama.


## Tehnologije

**Backend:**
- Java 24
- Spring Boot 3.5
- Spring Data JPA
- PostgreSQL
- Hibernate
- Spring Validation

**Frontend:**
- Angular 20
- Angular Material
- TypeScript
- RxJS
